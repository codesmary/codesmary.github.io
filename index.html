Ukulele Tab Generator

This tool allows for the generation of novel ukulele tabs based on a given number of artists to be used as inspiration. This is achieved by scraping UltimateGuitar.com for the discography of these artists and training a neural network on the collected tabs. This was created as a personal side project.

Parametric Plotter

This plotter contains tools for graphing three-dimensional parametric surfaces in a scene that can be freely rotated. There's the option to define parameters for built-in functions, as well as the ability to define custom equations. This was created as a personal side project.
